# Apple 26 AI Skill

English | [中文](#中文)

## English

`apple-26-ai-skill` helps coding agents implement Apple developer AI and coding-agent features introduced or expanded around WWDC2026.

It is designed for Claude Code, Codex, Cursor-style agents, and similar coding agents working on iOS, iPadOS, macOS, visionOS, watchOS, Swift, SwiftUI, and Xcode projects.

## 2026 Feature Set

- Xcode coding intelligence for chat and agent workflows.
- External agent access to Xcode through Xcode MCP and `xcrun mcpbridge`.
- Claude Agent and Codex workflows inside Xcode projects.
- Apple Foundation Models app features: structured output, guided generation, tool calling, streaming, context handling, and multi-session agentic flows.
- Private Cloud Compute-backed Apple Foundation Model flows with availability and fallback handling.
- Custom LLM providers through Foundation Models provider abstractions.
- `fm` CLI and Foundation Models Python SDK workflows for scripting, prototyping, and evaluation.
- Core AI on-device model deployment with `.aimodel` assets, model specialization, ahead-of-time compilation, Xcode inspection, and Instruments profiling.
- MLX or local Apple silicon model workflows when appropriate.
- Apple Dynamic Profiles and Instruments workflows for comparing model, prompt, tool, schema, privacy, fallback, and latency behavior.

## 中文

`apple-26-ai-skill` 用于帮助 coding agent 落地 WWDC2026 前后 Apple 新增或扩展的开发者 AI 与 coding-agent 功能。

它适用于 Claude Code、Codex、Cursor 类 agent，以及类似的自动编码工具；目标项目包括 iOS、iPadOS、macOS、visionOS、watchOS、Swift、SwiftUI 和 Xcode 项目。

- Xcode coding intelligence：面向 chat 和 agent workflow。
- 通过 Xcode MCP 和 `xcrun mcpbridge` 让外部 agent 访问 Xcode 能力。
- Claude Agent 和 Codex 在 Xcode 项目中的 agent workflow。
- Apple Foundation Models app 功能：结构化输出、guided generation、tool calling、streaming、context handling 和多 session agentic flow。
- Private Cloud Compute-backed Apple Foundation Model：包含 availability 和 fallback 处理。
- 通过 Foundation Models provider abstraction 接入自定义 LLM provider。
- `fm` CLI 和 Foundation Models Python SDK：用于脚本化、原型和评测。
- Core AI 端侧模型部署：`.aimodel` 资产、model specialization、ahead-of-time compilation、Xcode 检查和 Instruments profiling。
- 需要时使用 MLX 或本地 Apple silicon 模型 workflow。
- Apple Dynamic Profiles 和 Instruments workflow：对比 model、prompt、tool、schema、隐私模式、fallback 和 latency 行为。
